  2.ThinkPHP

  ThinkPHP是一个免费开源的，快速、简单的面向对象的轻量级PHP开发框架，是为了敏捷WEB应用开发和简化企业应用开发而诞生的。

  3.检测tp5.0 的环境

    PHP >= 5.4.0
    PDO PHP Extension
    MBstring PHP Extension
    CURL PHP Extension

  4.TP5.0的安装

  <1> 源代码包

      a. 去ThinkPHP官网下载（www.thinkphp.cn）下载完整版的Tp5.0

      b. 解压压缩包

      c. 将压缩文件拷贝到 网站根目录（www）

      d. 重命名为tp5

      e. 地址栏访问  http://localhost:8888/tp5/public/


  <2>composer安装

     a. 官方网站  https://www.phpcomposer.com/

        Composer是 PHP 世界里用于管理项目依赖的工具。类似于Linux中的yum源

     b. 下载地址

        https://getcomposer.org/download/

  <3>git安装

     首先克隆下载应用项目仓库
     git clone https://github.com/top-think/think tp5

     然后切换到tp5目录下面，再克隆核心框架仓库：
     git clone https://github.com/top-think/framework thinkphp
